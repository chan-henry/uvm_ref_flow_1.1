* Title       : Package for the GPIO bus protocol.
* Name        : gpio
* Created     : March 2011
* Version     : 1.02
* Description : 

This is the GPIO UVC.
GPIO is a basic point to point protocol and the UVC is implemented 
in Specman 'e'.The UVC uses basic UVM 'e' constructs.

* Directory structure

    gpio/e/        -- gpio UVC source files

    gpio/examples/ -- Verification Environment examples 

    gpio/demo.sh - Demo sim for GPIO UVC
