* Title       : Package for the SPI bus protocol.
* Name        : spi
* Created     : March 2011
* Version     : 1.02
* Description : This is the SPI UVC.

                in Specman 'e'. The UVC uses basic UVM 'e' constructs.

* Directory structure

  spi/e/        -- spi UVC source files

  spi/examples/ -- Verification Environment examples 

  spi/demo.sh   -- Demo sim for SPI UVC
