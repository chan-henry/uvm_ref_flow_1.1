* Title       : Package for the UART protocol.
* Name        : uart
* Created     : March 2011
* Version     : 1.02
* Description : This is the uart UVC which can support different types of uart 
                frames , it can drive as well monitor transactions at 
		uart interface.It is implemented in 'e' and uses basic 
                UVM 'e' constructs.

* Directory structure

    uart/e/        -- uart UVC source files

    uart/examples/ -- Verification Environment examples 

    uart/demo.sh   -- Demo sim for UART UVC
